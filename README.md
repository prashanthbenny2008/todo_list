## How to run

1. Clone the project or download/extract the project
2. Create .env file in the root directory or rename `.env.example` file as `.env`
3. Create a database for the app.
4. Change the title and database credentials in the .env file (DB_NAME as specified in step 3)
5. Run `php artisan migrate` in terminal to create the neccessary tables.
